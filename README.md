# Pädagogik-Olympiade 2027

Statische Website zur Pädagogik-Olympiade 2027 an der Universität Bielefeld. Gehostet via GitHub Pages.

## Struktur

- `index.html` — Landingpage
- `impressum.html` — Impressum & Datenschutz (Anker: `#datenschutz`)
- `.nojekyll` — verhindert Jekyll-Processing auf GitHub Pages
- `p-dagogik-olympiade-reloaded/` — Original-Handoff aus Claude Design (Referenz)

## Lokal entwickeln

```sh
python3 -m http.server 8000
# → http://localhost:8000
```
